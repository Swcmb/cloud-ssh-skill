# SSH 云服务器连接技能

在 Trae Work 环境下安全连接云服务器的完整指南和操作流程。

## 功能特性

- 🔑 密钥管理：生成和管理 Ed25519 密钥对
- 🌐 网络配置：处理代理环境下的 SSH 连接
- 🛡️ 安全配置：服务器端安全最佳实践
- 🐛 故障排查：常见问题诊断和解决方案

## 安装

```bash
npx skills add https://github.com/Swcmb/cloud-ssh-skill --skill ssh-connect
```

## 使用

安装后，当你需要连接云服务器时，该技能会自动提供指导和帮助。

## 连接流程

1. **生成密钥**：在本地生成 Ed25519 密钥对
2. **配置服务器**：将公钥添加到服务器的 authorized_keys
3. **开放端口**：在云服务商安全组中开放端口 22
4. **配置代理**：在 Trae Work 环境中配置 ProxyCommand
5. **连接验证**：测试 SSH 连接

## 安全最佳实践

- 使用 Ed25519 密钥（更安全、更快）
- 禁用密码认证
- 限制 root 登录
- 使用 fail2ban 防止暴力破解
